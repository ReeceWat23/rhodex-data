# Contracts (Modeler + Live Globe)

Run commands from the **contracts** folder:

```bash
cd contracts
npm install
npm run dev
```

- **`npm run dev`** — Start dev server (Vite). Open the URL shown (e.g. http://localhost:5173).
- **`npm run build`** — Production build (output in `dist/`).
- **`npm run preview`** — Serve the production build locally.

## One-liner from repo root

```bash
cd Rhodex/contracts && npm install && npm run dev
```
